jQuery Leaves

This jQuery plugin will provide beautiful decorations on your website: falling and rotating interactive leaves. If you click on a leaf, it will fall; if you click on a falling leaf, magic will happen. The plugin is cross-browser, mobile-optimized and very easy to extend.

Setup

First you need to include the jQuery library, since Leaves is a plugin. You can download it from the jQuery website or link it directly from the Google CDN.

<script src="//ajax.googleapis.com/ajax/libs/jquery/3.1.1/jquery.min.js"></script></pre>

You will also need the jQuery rotate plugin, which you can insert into your page with the code below:

<script src="js/jquery.rotate.js"></script>

Secondly, you need to include the jQuery Leaves Javascript and the CSS file, which you can do it by adding the code below to your page.

<script src="js/jquery-leaves.js"></script>
<link rel="stylesheet" type="text/css" href="css/jquery-leaves.css" />

As the last step, you create an instance of the plugin object, which you can store in a variable for later use.

var tree = new catLeaves({
    leaves: 30,
    maxY: -10,
    multiply_on_click: true,
    multiply: 2,
    infinite: true,
    speed: 4000
});

This way, when you want to add more leaves, you can do:

tree.add(8);

Settings

Name Default Description
leaves 50 Number of leaves to spawn initially.
max_y 100 Spread size.
multiply_on_click true Whether to multiply the leaf when the user clicks on it.
multiply 1 Number of images to spawn when clicking on a leaf.
infinite true If you want the leaves to never stop falling.
speed 3000 Speed of the falling leaf animation.
num_images 8 Total number of images in the leaves folder.
folder img/leaves The folder containing the leaf images.

Methods

Name Description
add() Add the specified amount of leaves.

Add More Leaves

Code:

<a href="#" class="addLeaf">Add More Leaves</a>

var tree = new catLeaves({
    leaves: 30,
    max_y: -10,
    multiply_on_click: true,
    multiply: 2,
    infinite: true,
    speed: 4000
});
$('body').on('click', '.addLeaf', function() {
    tree.add(8);
    return false;
});
